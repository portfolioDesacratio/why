# why? — brand site

Minimalist streetwear brand landing page.

## Setup

1. Push to GitHub
2. Enable GitHub Pages in repo settings (source: main branch, root folder)
3. Update `script.js` — replace `WEBHOOK_URL` with your webhook.site URL

## Tracker

For data collection, either:
- **webhook.site** (easiest): go to https://webhook.site, copy your URL, paste in script.js
- **PHP backend**: upload `track.php` to a free PHP host, update WEBHOOK_URL in script.js
